# Local Security Evidence Template

## Purpose

This template records free local security evidence without treating it as an
external penetration-test attestation.

Use it for internal evidence packets from Bandit, Safety, pip-audit, npm audit,
Trivy, CodeQL, Semgrep, ZAP, Nuclei, workflow policy tests, or release-artifact
checks.

## Boundary

This record is internal security evidence only. It must not be described as a
third-party penetration test, external attestation, compliance certification, or
commercial readiness proof.

Keep H5 open until an external tester or assessment owner supplies a dated
report or signed attestation with scope, method, severity summary, remediation
mapping, retest status, and an accountable claim owner.
