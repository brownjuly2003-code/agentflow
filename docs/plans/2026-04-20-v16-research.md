## TASK 1 — Исследовать интеграционные паттерны Langchain/LlamaIndex

**Цель:** понять какой API surface ожидают агенты от data layer.

### Что исследовать

1. **Langchain Tool patterns:**
   - Как кастомные tools интегрируются (`@tool` decorator, `BaseTool`, `StructuredTool`)
   - Типовые tools для data access: SQL, vector search, REST API
   - Как Langchain агенты вызывают tools при reasoning

2. **LlamaIndex data connectors:**
   - `LoadAndIndex`, `VectorStoreIndex`, `QueryEngine` — что из этого применимо к AgentFlow
   - Streaming data sources для LlamaIndex
   - Как LlamaIndex агенты делают data freshness checks

3. **Emerging patterns (2025-2026):**
   - MCP (Model Context Protocol) — стал стандартом или нет
   - Tool-use best practices от Anthropic, OpenAI
   - Agent memory patterns и их связь с real-time data

### Источники

- docs.langchain.com/docs
- docs.llamaindex.ai
- Anthropic MCP: modelcontextprotocol.io
- GitHub: langchain/langchain, run-llama/llama_index (recent issues, PRs)
- HackerNews, r/LocalLLaMA обсуждения за последние 3 месяца

## TASK 3 — Customer Discovery Questions

**Цель:** подготовить вопросник для 5 интервью с потенциальными ICP (dev teams 5-50, building AI agents).

### Что написать

Вопросы в 5 блоков (по 3-5 вопросов в каждом, открытые не closed):

1. **Current pain** (без наводки на AgentFlow)
   - "Расскажи про последний случай когда ваш агент дал неправильный ответ из-за stale data"
   - "Как вы сейчас решаете freshness?"

2. **Technical constraints**
   - "Какие latency targets?"
   - "Что не устроило в существующих решениях (если пробовали)?"

3. **Integration reality**
   - "На чём у вас агенты? Langchain / custom / LlamaIndex / что-то ещё?"
   - "Как агенты получают данные сейчас?"

4. **Buying signals** (осторожно, в конце)
   - "Как у вас обычно закупаются dev tools?"
   - "Если бы идеальное решение существовало, что бы в нём было критично?"

5. **Willingness-to-pay probes**
   - "Сколько сейчас тратите на data infra для агентов?"
   - "Какая-нибудь особенность стоит $X/месяц?"

### Do / Don't

- DO: открытые вопросы, factual ("расскажи о последнем случае..."), молчать после вопроса
- DON'T: наводящие ("вам же нужна sub-30s freshness?"), пичить AgentFlow, обещать фичи
